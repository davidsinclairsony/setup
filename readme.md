# setup
install as much as possible through cli

## install
```
cd ~/Downloads/setup
chmod +x ./cyclo.sh; ./cyclo.sh
chmod +x ./nodessertro.sh; ./nodessertro.sh
chmod +x ./sony.sh; ./sony.sh
```

## post install
- run `rm -rf ~/Downloads/setup`
- configure: dropbox, firefox, transmission
- add ssh keys
